# Unit Testing (단위 테스트) Kotlin 예제

단위 테스트(생산성과 품질을 위한 단위 테스트 원칙과 패턴)에 대한 Kotlin 예제 

## Prerequisite
* JUnit 5
  * ```testImplementation("org.assertj:assertj-core:3.3.0")```
* Run Configuration에 **"Use tracing"** 활성화
